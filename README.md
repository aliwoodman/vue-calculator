# vue-calculator

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn start
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

### Firebase deployment
```
yarn build
firebase serve (optional - check changes)
firebase deploy -m [DEPLOY-MESSAGE]
```
